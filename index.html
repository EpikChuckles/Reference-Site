HTML References

HTML Tables

Table Tags

Tag Name Description
<table> Table The wrapping element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing the actual table data.
<tr> Table Row The container that holds the data for each column in a row.
<th> Table Header Cell Holds the data for the column headers in the Table Head.
<td> Table Data The data held in a column in the Table Row.
<tfoot> Table Foot The set of rows defining the footer of a table. Usually for results or conclusions.

Table Attributes

Attribute Name Description
<td colspan =""> Column Span Defines how many columns a td element should span.
<td rowspan=""> Row Span Defines how many rows a td element should span.


CSS References

CSS Display and Positioning

Display Properties

'display:"---";'|Dictates how space is shared with other elements.
Name Description
'inline' Element does not require a new line. Height and width can not be adjusted.
'block' Element will be on it's own line. Can contain other block or inline elements.
'inline-block' Element is displayed 'inline', but height and width can be adjusted.
'grid' Element is displayed 'block' with inner content in grid layout.
'flex' Element is displayed 'block' with inner content in flex layout.
'none' Element and all children will be removed.

Default Displays

Name Description
'inline' <span>, <input>, <button>, &<img>
'block' <div>, <p>, <h1> thru <h6>, <ul>, <li>, &<canvas>

Notes

Name Description
'inline-block' May also be used with grid, flex or table as these are block-level.
'none' To make an item invisible but still take up space on the page use 'visibility:"---";'

Positioning Properties

'position:"---";'|Used to change the default position of an element.
Name Description
'static Default position. Offset properties have no effect.
'relative' Default position. Can be moved using offset properties.
'absolute' Behaves as relative, but is ignored be all other elements on the page.
'fixed' Behaves as relative, but is not affected by user scrolling through the browser.
'sticky' Behaves as relative, until certain scroll point is met, then is 'fixed'.

Offset Properties

'top: 20px;'|As an example.
Name Description
'top' Moves the element down. Space above element.
'bottom' Moves the element up. Space below element.
'left' Moves the element right. Space to the left of element.
'right' Moves the element left. Space to the right of element.
'z-index' Moves element forward or backward. 'z-index: 2;' will be in front of 'z-index: 1;'.

Float Properties

'float:"---";'
Name Description
'left' Moves the element as far left as possible.
'right' Moves the element as far right as possible.

Clear Properties

'clear:"---";'|Specifies how elements should behave when they bump into each other on the page.
Name Description
'left' The left side of the element will not touch any other element within the same containing element.
'right The right side of the element will not touch any other element within the same containing element.
'both' Neither side of the element will touch any other element within the same containing element.
'none' The element can touch either side.

Notes

Name Description
Consider an inline element such as an <img> nested within a <p>. Setting the float value to ‘left’ will put the image on the left side & the text to the right, wrapping to the full line below where the image ends.
